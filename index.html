fReady by matt holden

fReady (or, "Freddy") is a light wrapper to Facebook's Javascript SDK, with zero dependencies.

Install

  • Add fReady.js to your project!
  • Include a reference to the Facebook's SDK in your document head. (If omitted, fReady will load the SDK for you.)
  • Somewhere beneath the body element, add: 
    <div id="fb-root"></div>
<script type="text/javascript" src="/path/to/fReady.js"></script>

Usage

Start by calling fReady.init()

// Initialze Facebook with your appID.
fReady.init({
  appId  : YOUR APP ID HERE
  //Pass other FB.init() properties here if you like
});

Next, pass fReady() one or more functions to be called after the user's login status has been determined. If the user is connected to your app, their user info will also be pre-fetched, so you can access their personal information synchronously.

A simple demo: who is the current user? (Tell me my name!)

fReady(function(){
  if (fReady.isLoggedIn()){
    var your_name = fReady.getMe().name;
    alert(your_name) //Your Name
  }
});

fReady also supports pre-defined callbacks to be executed in response to changes in your user's login status. Much more functionality is on its way, all aiming to streamline common Platform tasks. It is currently in a pre-pre-pre alpha state, and documentation is ongoing. Feel free to help!
