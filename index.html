fReady by matt-holden

fReady is a light wrapper to Facebook Platform's Javascript SDK.

Dependencies

None!

Install

Just add fReady.js to your project and go!

Usage

In it's simplest form -- This is the code running on this page!


// Queue up a function to execute once we know something about the user.
fReady(function(response){
	alert( fReady.me.name ); //Your name is foo
});

// Somewhere else in the code...
// Initialze Facebook Just like you always have, with two optional properties at the end..
fReady.init({
	appId		: YOUR APP ID HERE,
	channelURL	: '//WWW.YOUR_DOMAIN.COM/channel.html', // Channel File
	status		: true, // check login status
	cookie		: true, // enable cookies to allow the server to access the session
	oauth		: true, // enable OAuth 2.0
	xfbml		: true,  // parse XFBML
	
	//notice the two extra properties!
	queryScopeFirst : true,
	scope		: scope
});
